Privacy Policy

This page explains what information Time Management Coach processes, where data is stored, and what third-party services are used when optional features are enabled.

Last updated: March 18, 2026

Local-first Tasks and preferences are primarily stored in your browser.
Optional sign-in Google account data is used only if you choose Google login/sync.
AI services Prompts may be sent to OpenAI or Gemini when AI coaching is used.

1. Information We Process

The app can process the following categories of information:

  • Task content you create (title, notes, due date/time).
  • Chat prompts/messages you send to the AI coach.
  • Preference data such as theme and auto-task toggle settings.
  • Basic Google profile info (name, avatar, email) only if you sign in with Google.

2. How Data Is Used

  • To run core app features (chat, tasks, timer, and reminders).
  • To improve continuity in AI conversations using short in-browser context memory.
  • To sync tasks with Google Tasks only when you explicitly enable Google sync.

3. Storage and Retention

By default, this app is frontend-only and stores data in browser storage (for example localStorage). Data remains on your device until you clear it or reset app data.

Default: local storage Do not store sensitive secrets in chat prompts

4. Third-Party Services

When enabled, the app can connect to these services:

  • OpenAI API for coaching responses.
  • Google Gemini API as fallback AI provider.
  • Google Identity Services and Google Tasks API for authentication and task sync.

Each provider has its own privacy policy and data handling terms.

5. Security Notice

This project is designed for educational and productivity use. If deployed publicly, API keys should never be exposed in client-side code. Use a secure backend proxy for production-grade secret management.

6. Your Choices

  • Use the app without Google login.
  • Disable auto-add task behavior from the AI coach.
  • Reset conversation memory from the app controls.
  • Clear browser storage to remove locally saved data.

7. Contact

For privacy questions, contact ventures.skyworld@gmail.com.
